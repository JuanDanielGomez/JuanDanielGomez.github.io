Geolocated vehicle surveillance

This project is developped in HTML, CSS and JS for the web page and Java for the mobile application. It shows a vehicle position and other pertinent data in real time as well as previous data stored in a data base. The data is taken with the help of an OBD device and send to a remote socket via the mobile app. It displays the data in a graphic way, using a map to show the vehicle current position.

Cellular automaton

Simulator written in java for different cellular automatons like Conway's game of life, the game of immigration, Schelling model and a group of Boids with a simulated behavoir of preys and predators.

*template code provided by Grenoble INP - Ensimag, the code can not be showed for university politics reasons.

Conception of an operating system

This project is written in C. It covers the development and implementation of a operating system core. Managing topics such as: timed interruptions (using machine clock), I/O ports (screen), dynamic creation and execution of processes and managing their ordering as well.

*template code provided by Grenoble INP - Ensimag, the code can not be showed for university politics reasons.

Video player

This project is written in C, it manages threads to ensure the reproduction of a video and sound. Synchronisation tools are used such as mutex to ensure the correct reproduction of the video. Manages different threads of consumers and producers syncrhonised.

*template code provided by Grenoble INP - Ensimag, the code can not be showed for university politics reasons.

Shell

This is a project written in C that tries to emulate an unix terminal with basic commands implemented using fork for creating new processes and file decryptors commands like fd, open, close and pipe.
